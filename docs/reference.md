# API reference

The API surface is identical across all three targets — one F# source tree
produces every package — so the generated **F# API reference** is the
authoritative reference for .NET, JavaScript/TypeScript, and Python alike.
Member names, parameters, and defaults carry over one-to-one; only the async
wrapper differs per language (`Async` / `Promise` / `awaitable`).

**[Open the F# API reference](/DataHubClient/fsdocs/reference/index.html)** —
generated by [fsdocs](https://fsprojects.github.io/FSharp.Formatting/) from the
XML documentation comments in the source tree.

!!! note
    The reference is generated into `site/fsdocs/` by `./build.sh BuildDocs`;
    when previewing the guide with `mkdocs serve` alone, the link above 404s.
